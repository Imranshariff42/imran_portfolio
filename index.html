Customer Purchase Behavior
Analytics

Python + SQL + Power-BI

Problem: Understand customer purchasing patterns and identify trends that influence sales performance and customer retention.

Approach: Extracted and queried customer transaction data using SQL, performed data cleaning and exploratory analysis in Python, and built interactive Power BI dashboards to visualize purchase trends, customer segments, and key performance metrics.

Key Insights: Identified high-value customer segments and repeat purchase behavior Analyzed purchase frequency and seasonal buying trends Highlighted product categories contributing most to overall revenue

OLA Rides Analytics Report

SQL + Excel + Power-BI

Problem: Analyze ride data to understand demand patterns, identify peak usage periods, and evaluate key operational metrics.

Approach: Used SQL Server to query and aggregate ride data, prepared and cleaned the dataset in Excel, and built a Power BI dashboard to visualize ride trends, revenue metrics, and ride distribution across different time periods.

Dataset: Ride dataset containing 100K+ ride records including booking details, ride timings, revenue information, and location data.

Key Insights: Identified peak ride demand during specific hours and days Highlighted high-demand time slots contributing most to ride volume Analyzed revenue distribution across different ride categories

Business Value: Demonstrates how ride demand analysis can help ride-hailing platforms optimize driver allocation, improve service availability, and support data-driven operational planning.

Pizza Sales Report

SQL + Power-BI

Problem: Analyze pizza sales data to understand revenue trends, product performance, and peak ordering periods.

Approach: Used SQL Server 2022 to query and aggregate sales data, performed data transformation and analysis, and developed an interactive Power BI dashboard to visualize revenue trends, product popularity, and sales distribution across time.

Dataset: Sales dataset containing 40K+ transaction records including order details, product categories, pricing, and order timestamps.

Key Insights: Identified top-performing pizza categories contributing most to revenue Analyzed peak sales hours and high-demand days Evaluated order distribution across product types and sizes

Business Value: Demonstrates how sales analysis can help businesses understand customer preferences, optimize product offerings, and improve inventory and marketing strategies.

AI Chatbot for Country
Capital Queries

Machine Learning + NLP + Python

Problem: Build a system capable of automatically answering user queries related to country–capital pairs using natural language input.

Approach: Developed an NLP-based chatbot using Python and integrated it with a Telegram interface to interact with users. The project involved preprocessing text data, training a machine learning model on a country–capital dataset, and implementing query interpretation to return accurate responses.

Dataset: Country–capital dataset containing mappings between countries and their respective capitals used for model training and response generation.

Outcome: Successfully built a chatbot capable of responding to user queries with ~90% accuracy, demonstrating how natural language processing and machine learning can automate simple information retrieval tasks.
